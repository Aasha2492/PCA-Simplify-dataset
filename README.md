# PCA-Simplify-dataset
This project demonstrates PCA for reducing dataset dimensions. It explores variance retention, scaling effects, and visualizations using Iris and Digits datasets with Scikit-learn and Matplotlib.
This project demonstrates the use of Principal Component Analysis (PCA) for dimensionality
reduction using Scikit-learn. It includes multiple exercises to understand variance, scaling effects,
and visualization techniques.
**       Exercises Overview:
**1. Keep Only PC1: Reduce dataset to one principal component and observe variance.
2. Digits Dataset: Determine number of components required for 95% variance.
3. Without Scaling: Analyze PCA results without StandardScaler.
4. 3D Visualization: Visualize data using three principal components.
Technologies Used:
Python, NumPy, Pandas, Matplotlib, Scikit-learn
      **Conclusion:
**PCA helps reduce dimensionality while preserving important patterns in data. This project highlights
its practical applications in data preprocessing and visualization.
